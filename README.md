🚖 Uber Rides Analysis Dashboard:

📌 Project Overview:
This project is a Power BI dashboard developed to analyze Uber ride booking data and provide insights into ride performance, booking trends, cancellations, 
and customer activity. The dashboard helps stakeholders monitor key business metrics and make data-driven decisions.

🎯 Objectives:
Analyze Uber ride booking patterns.
Track completed, cancelled, and incomplete rides.
Monitor booking trends and ride performance.
Visualize key business metrics through interactive dashboards.

📊 Dashboard Features:
Home Dashboard
Total Bookings
Total Booking Value
Average Ride Distance
Revenue generated by vechicle every month/Quatar
Completed rides by vechicle every month/Quatar
Ride Volume Trends
Vehicle Type Analysis
Overview Dashboard
Completed Rides Analysis
Cancelled Rides Analysis
Incomplete Rides Analysis
Booking Status DistributionRevenue Insights
Interactive Filters and Slicers

🛠️ Tools & Technologies:
